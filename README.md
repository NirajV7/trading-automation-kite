# Quantitative Trading System: Zerodha Kite Unified Terminal

This is a clean, Fyers-free trading terminal designed to run execution and data logging on Zerodha Kite APIs.

## Directory Structure
*   `backend/`: Python server code running KiteTicker streams, calculations, order execution, and FastAPI.
*   `frontend/`: Node.js/Electron client application running on MacBook Air.
*   `data/`: Watchlists, tokens, and active trades state cache.
*   `logs/`: Executions and live stream logs.
